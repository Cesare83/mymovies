myMovies Documentation

API description


Method Type URL Description Request body Response body
GET /movies Get a list of all movies - JSON object with movie list
GET /movies/:title Get a single movie by title - JSON object with following keys:
  • title
  • description
  • genre
  • director
  • image
GET /genres/:name Get a genre description by name - JSON object with following keys:
  • name
  • description
GET /directors/:name Get a single director by name - JSON object with following keys:
  • name
  • bio
  • birth
  • death

  • username
